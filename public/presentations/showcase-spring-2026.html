Why this program

  • Agentic AI coding tools are already in library work
  • Tasks that once required scripting are now within reach without it
  • A confident-looking output can still be wrong
  • CUNY does not provide access to these tools, so participants paid for their own subscriptions

Have you used these?

  • ChatGPT, Claude, Copilot, Gemini
  • You type something, it “types” back
  • Predicting the next word over and over
  • Trained on a lot of text, learned patterns
  • Fluent, but not grounded

No tools available. The model can only estimate from training data.

Reaching for groundedness

  • Sometimes it searches the web
  • Sometimes it reads a file you uploaded
  • Sometimes it runs code

A tool is a function the system can call. “Search this,” “fetch that.” That's tool use.

Same question. This time it actually looks it up.

Agentic means it loops

  • Regular: you ask, the model answers (maybe with one tool call)
  • Agentic: a harness wraps the model in a loop. It runs the tool call, feeds the result back, prompts the model again
  • Stops when the task seems done, or when it gets stuck
  • Harnesses: Claude Code, Codex, Cursor

What that looks like

“Find recent books on music theory we don't already own.”

  • Searches WorldCat, gets 25 recent music theory titles with ISBNs
  • Filters to titles not already held by the institution
  • Checks Primo by ISBN to verify holdings
  • Fetches publisher sites for working links
  • Returns a list ready for ordering: title, author, publisher, ISBN, link

(Then the librarian checks whether the list is actually right.)
